## Getting Started

:::info
**Quick Start**
Begin your journey with Proper Labs by creating your VerseID.
:::

### Installation

```bash
npm install @properlabs/sdk
```

### Basic Usage

```javascript
import { VerseID } from '@properlabs/sdk';

const identity = new VerseID({
  network: 'mainnet'
});

await identity.connect();
```
